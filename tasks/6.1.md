## Description

Implement WorkflowEngine and WorkflowState in src/workflow/mod.rs and src/workflow/transitions.rs. WorkflowPhase: Idle, Running, AwaitingAnswer, PendingReview, Completed, Errored. process() matches (current_phase, msg) to advance state: StartTask -> Running + CreateSession, SessionCompleted -> next agent or PendingReview, AgentCompleted -> advance pipeline, AgentKickedBack -> validate and re-route, AgentAskedQuestion -> AwaitingAnswer, HumanAnswered -> Running + CreateSession, HumanApprovedReview -> Completed, SessionError -> Errored.

Tests: test_start_task_transitions_to_running, test_agent_completed_advances_pipeline, test_code_review_completed_transitions_to_pending_review, test_valid_kickback_accepted, test_invalid_kickback_rejected, test_question_pauses_workflow, test_human_answer_resumes_workflow, test_human_approved_completes, test_session_error_transitions_to_errored.
