## Description

Implement the two-phase line-oriented markdown parser in src/tasks/parser.rs. Phase 1 parses Key: Value metadata (Story, Task, Status, Assigned To) before the first ## heading. Phase 2 splits on ## headings and parses known sections (Description, Starting Prompt, Questions, Design, Implementation Plan, Work Log) structurally; unknown sections are collected as extra_sections for round-trip fidelity.
