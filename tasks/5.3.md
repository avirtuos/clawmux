## Description

Implement EventStreamConsumer in src/opencode/events.rs. Connects to GET {base_url}/global/event via reqwest_eventsource, maps OpenCodeEvent variants to AppMessage values using a session_map (Arc<RwLock<HashMap<String, (TaskId, AgentKind)>>>). Events for unknown sessions are silently ignored. Reconnects on connection loss with exponential backoff (1s to 30s cap). Run as a tokio::task.

Tests: test_event_routing_known_session, test_event_routing_unknown_session, test_session_map_insert_and_remove.
